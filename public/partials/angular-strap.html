Modal directives/modal.js

Overview and examples

Fetches an external html partial (or an inline ng-template) and populates the modal with its content.

Use any button/link to trigger a modal by appending a bs-modal attribute.

Both $scope.show() and $scope.dismiss() are available inside the modal to toggle its visibility. 

modal: {{modal | json}}

<!-- Button to trigger modal -->
<button type="button" class="btn" bs-modal="'partials/modal.html'">...</button>

<!-- Modal (external ./partials/modal.html or inline ng-template) -->
<div class="modal-header">
  <button type="button" class="close" data-dismiss="modal" aria-hidden="true">Ã—</button>
  <h3>Modal header</h3>
</div>
<div class="modal-body">
  <p>{{modal.content}}</p>
  <br />
  <pre>2 + 3 = {{ 2 + 3 }}</pre>
</div>
<div class="modal-footer">
  <button type="button" class="btn" ng-click="dismiss()">Close</button>
  <button class="btn btn-primary" ng-click="modal.saved=true;dismiss()">Save changes</button>
</div>
Heads up! You can use ng-controller="MyModalCtrl" along bs-modal="..." to spawn a controller with a custom scope for your modal.

Navbar directives/navbar.js

Overview and examples

Performs regular expression tests with the data-match-route attribute when $location.path() gets updated.

Toggles the active class for you. Trigger the behavior with a bs-navbar attribute.

Title
  • Home
  • Page One
  • Page Two 
<div class="navbar" bs-navbar>
  <div class="navbar-inner">
    <a class="brand" href="#">Title</a>
    <ul class="nav">
      <!-- You can use regular expressions -->
      <li data-match-route="/(:?navbar)"><a href="#/navbar">Home</a></li>
      <li data-match-route="/page-one"><a href="#/page-one">Page One</a></li>
      <li data-match-route="/page-two.*"><a href="#/page-two/sub-a">Page Two</a></li>
    </ul>
  </div>
</div>

Tooltip directives/tooltip.js

Overview and examples

Parses the content passed to the directive & displays a tooltip with it.

Use any button/link to trigger a popover menu by appending a bs-popover attribute.

Both $scope.show() and $scope.dismiss() are available inside the tooltip to toggle its visibility. 

tooltip: {{tooltip | json}}

Tight pants next level keffiyeh you probably haven't heard of them. Photo booth beard raw denim letterpress vegan messenger bag stumptown. 

<!-- Basic tooltip on hover -->
<a href="#" bs-tooltip="tooltip.title">...</a>
<!-- Button to trigger tooltip on click -->
<button type="submit" class="btn" data-trigger="click" bs-tooltip="tooltip.title">...</button>
<!-- Force close others with data-unique attribute -->
<input type="text" data-trigger="focus" data-unique="1" bs-tooltip="tooltip.title"/>

Popover directives/popover.js

Overview and examples

Fetches an external html partial (or an inline ng-template) and populates the popover with its content.

Use any button/link to trigger a popover menu by appending a bs-popover attribute.

Both $scope.show() and $scope.dismiss() are available inside the popover to toggle its visibility. 

popover: {{popover | json}}

<!-- Button to trigger popover -->
<button type="button" class="btn" bs-popover="'partials/popover.html'">...</button>

<!-- Force close others with data-unique attribute -->
<button type="button" class="btn" data-unique="1" bs-popover="'partials/popover.html'">...</button>

<!-- Popover (external ./partials/popover.html or inline ng-template)  -->
<form name="popoverForm">
  <p ng-bind-html-unsafe="popover.content"></p>
  <br />
  <pre>2 + 3 = {{ 2 + 3 }}</pre>
  <div class="form-actions">
    <button type="button" class="btn" ng-click="dismiss()">Close</button>

  </div>
</form>
Heads up! You can use ng-controller="MyPopoverCtrl" along bs-popover="..." to spawn a controller with a custom scope for your popover.
